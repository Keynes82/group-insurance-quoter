# 团险报价助手 (Group Insurance Quoter)

为企业客户提供快速保费计算、方案模板和风控建议的 Web 应用。

## 问题

团险业务员每天遇到客户报价需求，但无法快速给出合理报价，影响销售效率。

## 用户

- 团险业务员
- 团险销售支持人员

## V1 功能

1. **保费计算**：根据企业客户信息 + 投保险种 + 保额，计算应交保费
2. **方案模板**：提供常见团险方案模板，快速套用
3. **风控建议**：基于客户行业和人员结构，给出风控建议
